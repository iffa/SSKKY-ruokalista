### Libraries used
SSKKY-ruokalista is made with these amazing libraries:

* Android support libraries
* Firebase (push notifications)
* Dagger 2
* RxJava (with RxAndroid and RxPreferences)
* Retrofit with Moshi
* ButterKnife
* Timber (with TimberTreeUtils)
* Mosby
* Materialize (for MD resources)
* Annotated Adapter
* Material Preference
* Shoot
* Crashlytics & Answers
* LeakCanary
